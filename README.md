Destriant
=========

A stock portfolio overview tool that runs entirely in the
browser. Spiritual successor of
[pfm](https://github.com/Artefact2/pfm). Released under the Apache
License 2.0.

Test it live at: <https://artefact2.gitlab.io/destriant/>

Build dependencies
==================

* make
* wget
* sassc

Getting started
===============

~~~
make
make host
~~~
